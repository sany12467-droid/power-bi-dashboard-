# Power-bi-Dashboard-
# Dashboard overview 
this project is the comprehensive analysis of the performance of "grameenphone" stock tradwd on Dhaka Stock Exchange . this dashboard can be used to analyze the performance of the stock from 10 feb, 2025 to 10 feb, 2026.investors can make decisions regarding their investment into this stock.
   # Key features
   1.KPIs for average high, low open and closeing prices.
   
   2.Quarterly aggregated price comparison.
   
   3.Daily aggregated trend analysis
   
   4.Monthly volume anlysis.
   
   5.Combined monthly price + volume chart.
   
   6.Monthly volume analysis.
   
   # Tools and skills used
   1.Power BI Desktop: Data modeling, DAX measures, and Visualization.
   
   2.Power Query: Data cleaning and transformation.
    
   3.Excel/CSV: Raw data source.
